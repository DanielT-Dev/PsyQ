| Field                 | Content                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **ID and name**       | **UC-1: Take a Personality Quiz**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **Primary actor**     | User                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| **Secondary actors**  | Quiz Analysis System                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| **Description**       | A User accesses the psychology-based quiz application, selects a quiz related to personality, thinking patterns, or behavior, answers a set of questions, and receives psychological insights based on their responses.                                                                                                                                                                                                                                                                                                                                             |
| **Trigger**           | The User chooses to start a quiz from the application interface.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| **Preconditions**     | **PRE-1.** User has opened the application. <br> **PRE-2.** Quiz database is available. <br> **PRE-3.** User is logged in if they want results saved to their account.                                                                                                                                                                                                                                                                                                                                                                                              |
| **Postconditions**    | **POST-1.** User responses are stored in the system. <br> **POST-2.** The system generates psychological analysis results. <br> **POST-3.** Results are displayed to the user and optionally saved in their profile.                                                                                                                                                                                                                                                                                                                                                |
| **Normal flow**       | **1.0 Take a Personality Quiz** <br> 1. User requests to view available quizzes. (see 1.0.E1) <br> 2. System displays a list of quizzes. <br> 3. User selects a quiz. <br> 4. System displays the first question. <br> 5. User answers the question. (see 1.1) <br> 6. System displays the next question. <br> 7. Steps 5–6 repeat until all questions are answered. <br> 8. User submits the quiz. (see 1.2) <br> 9. System analyzes answers. <br> 10. System displays personality insights and explanations. <br> 11. System saves results in the user’s history. |
| **Alternative flows** | **1.1 Skip question** <br> User skips a question → System marks it unanswered and continues to next question. <br><br> **1.2 Review answers before submission** <br> User chooses to review answers → System shows all responses → User edits them → Return to step 8 of the normal flow.                                                                                                                                                                                                                                                                           |
| **Exceptions**        | **1.0.E1 No quizzes available** <br> 1. System informs the user that no quizzes are available. <br> 2a. User cancels → System ends use case. <br> 2b. User retries later → System restarts use case. <br><br> **1.2.E1 Unanswered questions** <br> 1. System warns user that some questions are unanswered. <br> 2a. User completes them → Return to step 5. <br> 2b. User confirms submission → System continues analysis. <br><br> **1.0.E2 Analysis error** <br> 1. System informs the user that analysis failed. <br> 2. User may retry later.                  |
